# Raspberry Pi Experiments

This section contains hands-on experiments using Raspberry Pi for hardware control, sensor integration, and communication testing.

The objective is to explore Linux-based embedded systems and understand the interaction between software and physical hardware components.

---

## Experiments Planned

- GPIO LED control
- Push button input
- DHT11 temperature sensor integration
- Ultrasonic distance sensor
- UART communication
- I2C device scanning
- SPI communication tests

Each experiment will include:
- Objective
- Hardware used
- Pin configuration
- Code implementation
- Observed results
