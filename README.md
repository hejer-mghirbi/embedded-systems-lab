# embedded-systems-lab
Structured hands-on experiments in embedded systems: Raspberry Pi, STM32, PIC16F, FPGA (VHDL), and communication protocols.
Platforms Covered :
-Raspberry Pi (Linux-based embedded systems)
-STM32 Microcontrollers (ARM Cortex-M)
-PIC16F Microcontrollers
-FPGA Design using VHDL
